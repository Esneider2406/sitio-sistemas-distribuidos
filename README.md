# Reflexión sobre la actividad

## 1. ¿Qué ventajas encuentras en trabajar directamente en la nube con GitHub?
Trabajar en la nube con GitHub permite acceder al proyecto desde cualquier lugar sin necesidad de instalar programas adicionales. Además, facilita el control de versiones, la colaboración en equipo y la publicación rápida del sitio web mediante GitHub Pages.

## 2. ¿Qué desafíos tuviste al crear el sitio sin usar herramientas locales?
Uno de los principales desafíos fue la falta de herramientas avanzadas como editores con autocompletado o previsualización en tiempo real. También puede ser más difícil detectar errores rápidamente y organizar archivos sin un entorno de desarrollo completo.

## 3. ¿Cómo podrías mejorar este sitio web en el futuro?
Se podría mejorar agregando un diseño más moderno y responsivo para dispositivos móviles, incorporando animaciones, mejorando la accesibilidad y añadiendo más contenido interactivo como formularios funcionales o integración con bases de datos.

## 4. ¿Qué aprendiste sobre Sistemas Distribuidos al crear este sitio?
Aprendí que los sistemas distribuidos permiten que múltiples computadoras trabajen juntas como un solo sistema, lo cual mejora la escalabilidad y la tolerancia a fallos. También comprendí cómo servicios como GitHub Pages funcionan como parte de una infraestructura distribuida para alojar sitios web.
